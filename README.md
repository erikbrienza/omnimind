# OmniMind 🧘‍♂️

OmniMind è un'applicazione di meditazione moderna e intuitiva sviluppata con **React** e **Vite**. Il design è minimalista, completamente **responsive** e pensato per offrire un'esperienza immersiva e rilassante. 🌿✨

## 🎵 Musica di sottofondo
Durante la sessione di meditazione, una **dolce melodia** accompagna l'utente per favorire il rilassamento. La musica parte automaticamente quando il timer viene avviato e si interrompe quando la sessione termina.

## 🚀 Funzionalità principali
- **Interfaccia moderna e reattiva** 📱💻
- **Timer di meditazione** con opzioni di pausa e reset ⏳
- **Musica di sottofondo** per un’esperienza immersiva 🎶
- **Navigazione fluida** con **React Router** 🔗

## 💡 Come avviare il progetto
1. **Clona la repository**
   ```sh
   git clone https://github.com/erikbrienza/omnimind.git
   ```
2. **Installa le dipendenze**
   ```sh
   npm install
   ```
3. **Avvia il server di sviluppo**
   ```sh
   npm run dev
   ```

🌟 **Rilassati e goditi la tua sessione di meditazione con OmniMind!** 🌟
